- **Выбрал коммит regression-ratchet (#82)** через @sxx hang-safe тестами в `policy/src/lib.rs` (overlay_case_integration + git-object-guard-proxy) **вместо** полного git clone теста — последний flaky и медленный. Manual-only гапы (#3 decide()-skip требует live IPC, #5 full clone требует сети) честно документированы.
- **Выбрал гнать E2E install сам в фоне** (`bmh0x0wi9` через Bash run_in_background) **вместо** делегирования агенту — потому что прошлые два sh-агента на 40-минутных install-прогонах зависали, не дав вердикта. Фоновый Bash надёжнее и notification приходит при завершении.
- **Выбрал стратегию «дожать Hermes user-mode + храповик + coherence-фиксы»** (с пересмотром после посылки пользователя про конечность ФС-поверхности) **вместо** ранее рассматриваемого bindflt-спайка. bindflt откладывается до момента, когда новая кочка реально окажется не лечимой user-mode.
